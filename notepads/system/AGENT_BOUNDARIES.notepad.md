# Agent Boundaries — Atlas Wealth

## Purpose
This notepad defines the explicit boundaries applied to AI agents.

## Permitted Capabilities
Agents may:
- Observe event streams
- Generate drafts and explanations
- Propose alternatives
- Run simulations
- Surface risks and anomalies

## Prohibited Capabilities
Agents may not:
- Issue advice
- Approve advice
- Accept on behalf of clients
- Execute transactions
- Modify evidence
- Escalate authority

## Design Intent
Agents augment human reasoning; they do not replace it.

This document is explanatory only.
