# Atlas Wealth — Pixel Review Checklist

This checklist must be completed on every UI PR.

## Geometry
- [ ] Sidebar = 256px
- [ ] Top bar = 56px
- [ ] Content padding = 24px
- [ ] Single scroll container only

## Typography
- [ ] Token-only font sizes
- [ ] Token-only line heights
- [ ] No inline styles

## Components
- [ ] Only approved primitives used
- [ ] No layout divs duplicating components

## Visual
- [ ] No colour drift
- [ ] No spacing drift
- [ ] No accidental responsiveness

## Verdict
- [ ] PASS
- [ ] FAIL
- [ ] DEFERRED (v2)
